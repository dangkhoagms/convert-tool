💡 Trước khi bắt đầu

Đảm bảo bạn có các file ảnh muốn chuyển đổi và kết nối internet ổn định. Quá trình chuyển đổi diễn ra hoàn toàn trên trình duyệt của bạn, không cần upload lên server.

Bước 1: Chuẩn bị file ảnh

Định dạng hỗ trợ Dung lượng tối đa Số lượng Ghi chú
JPG, JPEG 50MB/file Tối đa 100 file Phổ biến nhất
PNG 50MB/file Tối đa 100 file Hỗ trợ nền trong suốt
WebP 50MB/file Tối đa 100 file Định dạng mới, chất lượng cao
HEIC 50MB/file Tối đa 100 file Từ iPhone/iPad

Hướng dẫn từng bước

1
Truy cập ToolConverts.com
Mở trình duyệt và truy cập trang chủ
• Mở trình duyệt web (Chrome, Firefox, Safari, Edge)
• Nhập địa chỉ: toolconverts.com
• Chọn công cụ "Ảnh → PDF"
2
Chọn ảnh cần chuyển đổi
Có 2 cách để chọn ảnh
Cách 1 - Kéo thả:
• Kéo file ảnh từ thư mục vào vùng "Kéo thả ảnh vào đây"
• Có thể chọn nhiều file cùng lúc

Cách 2 - Click chọn:
• Click vào vùng upload
• Chọn file từ hộp thoại mở ra
• Giữ Ctrl (hoặc Cmd) để chọn nhiều file
3
Tùy chỉnh cài đặt PDF
Điều chỉnh theo nhu cầu của bạn
Khổ giấy: A4 (khuyên dùng), Letter, A3, Tự động
Chất lượng: Cao nhất, Tốt (khuyên dùng), Trung bình
Bố cục: 1 ảnh/trang, 2 ảnh/trang, 4 ảnh/trang
4
Sắp xếp thứ tự ảnh
Kiểm tra và điều chỉnh thứ tự
• Xem trước tất cả ảnh đã chọn
• Số thứ tự hiển thị ở góc dưới mỗi ảnh
• Click nút "×" để xóa ảnh không cần thiết
• Click "Xóa tất cả" để bắt đầu lại
5
Tạo và tải PDF
Hoàn thành quá trình chuyển đổi
• Click nút "Tạo PDF" màu xanh
• Theo dõi thanh tiến trình
• File PDF sẽ tự động tải xuống khi hoàn thành
• Kiểm tra thư mục "Downloads" của trình duyệt

💡 Mẹo và thủ thuật

Tối ưu hóa chất lượng

🎯 Lựa chọn chất lượng phù hợp

• Cao nhất: Dùng cho ảnh quan trọng, cần in ấn
• Tốt: Dùng cho hầu hết trường hợp, cân bằng chất lượng/dung lượng
• Trung bình: Dùng khi cần file nhẹ để gửi email

Tối ưu hóa bố cục

  • 1 ảnh/trang: Phù hợp với ảnh lớn, cần xem chi tiết
  • 2 ảnh/trang: Tiết kiệm giấy khi in, phù hợp ảnh khổ dọc
  • 4 ảnh/trang: Tạo album ảnh compact, xem tổng quan

Xử lý ảnh trước khi chuyển đổi

✅ Chuẩn bị ảnh tốt nhất

• Đổi tên file theo thứ tự (01, 02, 03...) để sắp xếp đúng
• Xoay ảnh về đúng hướng trước khi upload
• Nén ảnh quá nặng bằng công cụ khác nếu cần
• Loại bỏ ảnh trùng lặp hoặc mờ

Shortcuts hữu ích

Thao tác Cách thực hiện Lưu ý
Chọn nhiều file Ctrl + Click (Windows) / Cmd + Click (Mac) Trong hộp thoại chọn file
Chọn tất cả file Ctrl + A / Cmd + A Trong thư mục ảnh
Kéo thả nhiều file Chọn nhiều file rồi kéo cùng lúc Nhanh hơn chọn từng file

🔧 Khắc phục sự cố

Sự cố thường gặp

Nguyên nhân: File ảnh vượt quá 50MB

Giải pháp:
• Nén ảnh bằng công cụ khác trước khi upload
• Giảm độ phân giải ảnh
• Chuyển từ PNG sang JPG để giảm dung lượng
• Chia nhỏ thành nhiều lần chuyển đổi
Nguyên nhân: Thiếu RAM hoặc quá nhiều file

Giải pháp:
• Đóng các tab khác đang mở
• Giảm số lượng ảnh xuống dưới 50 file
• Restart trình duyệt và thử lại
• Sử dụng trình duyệt khác (Chrome khuyên dùng)
Nguyên nhân: Trình duyệt chặn download hoặc popup

Giải pháp:
• Cho phép popup/download cho toolconverts.com
• Kiểm tra thư mục Downloads
• Thử lại hoặc click chuột phải → "Save as"
• Tắt adblocker tạm thời
Nguyên nhân: Metadata EXIF của ảnh không được đọc đúng

Giải pháp:
• Xoay ảnh về đúng hướng trước khi upload
• Sử dụng phần mềm edit ảnh để "save as" lại
• Chụp ảnh ở hướng ngang thay vì dọc
• Loại bỏ metadata EXIF nếu có thể
